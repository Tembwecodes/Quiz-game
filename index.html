QUIZ TIME GAME

Game Instructions

The objective of the game is to answer all questions correctly to earn full points at the end of the game. The player reads the question and assesses for the correct options given. The player then picks the correct answer and waits for the next question to load. The game continues until all questions have been answered by the player. The player is then shown the score at the end of the game and is declared winner or loser. The game includes a variety of questions with multiple choice open-ended.
